# Reference Guide

## File & Folder Structure
- `demos/`: Demo environments
- `templates/`: Deployment templates
- `scripts/`: Automation scripts
- `site/`: Dashboard and metadata
- `docs/`: Documentation

## Key Scripts
- `gh-demo-secrets.sh`: Manage secrets
- `scaffold-demo.sh`: Scaffold new demos
